# App 1：AI 自動板擦機器人 ── Scratch 專案藍圖

## 專案說明

這個 Scratch 專案模擬「AI 自動板擦機器人」的工作情況。我用 Scratch 畫出機器人和黑板，讓大家可以像操作真的 App 一樣，按下按鈕、選擇要擦的區域，然後看機器人自己走過去把黑板擦乾淨。雖然 Scratch 裡沒有真的 Arduino，但我把程式的邏輯都用積木模擬出來了，讓評審老師更容易了解我的作品是怎麼運作的。

---

## 專案連結

> 請到 https://scratch.mit.edu 新增專案，命名為「AI板擦機器人模擬」

## 角色（Sprites）清單

### 角色 1：板擦機器人
- **造型列表**：待命、移動中、擦拭中、完成
- **初始位置**：x=-150, y=0
- **大小**：60%
- **功能**：接收廣播指令後，移動到黑板位置並來回擦拭

### 角色 2：黑板
- **造型列表**：乾淨、有字跡A區、有字跡B區、有字跡C區、全部有字跡
- **初始位置**：x=80, y=0
- **大小**：100%
- **功能**：根據機器人清潔的區域，切換成乾淨的造型

### 角色 3：AI 助手
- **造型列表**：一個（可愛機器人頭像）
- **初始位置**：x=200, y=150
- **大小**：40%
- **功能**：顯示說話泡泡，提示目前機器人的狀態

### 角色 4：按鈕組（共 4 個，每個是獨立 Sprite）
- **按鈕 A**：「擦 A 區」，位置 x=-180, y=-140
- **按鈕 B**：「擦 B 區」，位置 x=-80, y=-140
- **按鈕 C**：「擦 C 區」，位置 x=20, y=-140
- **按鈕全板**：「全板擦除」，位置 x=130, y=-140

### 角色 5：控制按鈕（共 2 個）
- **開始清潔按鈕**：位置 x=-180, y=-170，按下後廣播 `開始清潔`
- **暫停按鈕**：位置 x=0, y=-170，按下後廣播 `暫停`

## 背景

## 互動說明

| 操作 | 效果 |
|---|---|
| 點擊「擦 A 區」按鈕 | 機器人走到 A 區，黑板 A 區變乾淨 |
| 點擊「擦 B 區」按鈕 | 機器人走到 B 區，黑板 B 區變乾淨 |
| 點擊「擦 C 區」按鈕 | 機器人走到 C 區，黑板 C 區變乾淨 |
| 點擊「全板擦除」按鈕 | 機器人來回擦整塊黑板，全部變乾淨 |
| 點擊「暫停」按鈕 | 機器人立刻停下來，等待下一個指令 |
| 按空白鍵 | 重置場景（黑板回到有字跡狀態） |

> 提示：一次只能進行一個清潔任務。如果想換區域，先按「暫停」再選新的區域。

## 廣播訊號對照表（對應真實 Arduino 指令）

| Scratch 廣播 | 對應 Arduino 指令 |
|---|---|
| 清潔A區 | ERASE_REGION_A |
| 清潔B區 | ERASE_REGION_B |
| 清潔C區 | ERASE_REGION_C |
| 全板擦除 | ERASE_ALL |
| 開始清潔 | CLEAN_START |
| 暫停 | PAUSE_TASK |
| 全板清潔完成（動畫） | FIREWORK |
